What is this?

I'm trying to practice as much as I can using HTML, CSS, and JavaScript. So I created this web page to basically redirect you to the book exercises. I am also trying to familiarize myself with frameworks so, I'm using Bootstrap to help me with the design. In addition, I'm going to create a server using Spring Boot.

Spring boot is primarily going to help me send forms. Up to this point when we create forms we are not sending them anywhere. Now with Spring Boot the forms can be sent back to the server. I think this would give me good practice with forms and some backend programming
