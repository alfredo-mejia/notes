* Final Thoughts

You might be asking why is this book under the "Web" section if PHP is a programming language and MySQL is a database. Surely these technologies can be grouped under some other folder.
However, context matters, and PHP is an open-sourced scripting programming language that is mainly used in web development.
Simple web applications can be divided into 3 components: the frontend (what the browser sees such as HTML, CSS, and CSS), the backend (what the frontend interacts with), and a database.
MySQL will be the database we will be using. MySQL is open-source just like PHP but it is not required when building a web application. You may use any database of your choosing, open-source or not.
This can be Oracle, Postgres, or any other database. So going back to the original question, why is this book under the web section.
This is because after learning HTML, CSS, and JS we have learned how to build the frontend of web applications. Now we need to learn how to build the backend.
The backend of an application can use different technologies, it does not need to be PHP and MySQL. However, in our case, we will start with PHP & MySQL.
So, this book is in the web section because now we are going to build the backend of a web app, the backend will introduce new functionality to our web app than just using HTML, CSS, and JS.
